Introduction

C++ is a general-purpose programming language that was developed as an enhancement of the C language to include object-oriented paradigm. It is an imperative and a compiled language.

Hello World

To get started with writing C++, open DevC++ and write your first C++ code "Hello World" :

void main(){
cout<<"Hello World!";
return 0;
}
Variable

You can declare a variable like this:

int x = 10;

or like this:

int x;
x = 10;
Data types

C++ has many data types like:

  • Integer
  • Floating point number
  • Character
  • Void
  • String

For example:

int x = 10;
printf("%d",x);

We can see the output of this code is 10

But with

float y = 1/2;
printf("%4.3f",y);

The output will be 0.500
